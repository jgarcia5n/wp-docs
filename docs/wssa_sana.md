# Sana – Overview (Workday Integration)

---

## Sana Core

Workday-only edition of Sana Enterprise for HR and finance tasks, running solely on Workday data.

**Included capabilities:**
- Read and write actions in Workday through the Self-Service Agent
- Agentic chat with user history and citations
- Automated workflows that orchestrate HCM and FIN multi-step processes
- Enterprise search across Workday

**Licensing:** Automatically included when customer signs the Workday Flex Credits & Platform Entitlement Policy.

---

## Sana Enterprise

Unified front door for any business task using AI, connecting HR, finance, IT, collaboration tools, and all integrated systems.

**Included capabilities:**
- Read and write actions across core business systems
- Agentic chat with user history and citations
- Automated workflows across the enterprise tech stack
- Enterprise search across the full business suite
- Deep integrations to essential third-party systems
- Unified governance, permissions, and audit logs

**Licensing:** Additional SKU (not included with Flex Credits policy).

---

## Sana Capabilities Framework

| Capability | Description |
|---|---|
| **Find** | Instant access to company knowledge |
| **Act** | Take actions across applications and tools |
| **Build** | Generate ready docs, analyses, and dashboards |
| **Automate** | Run complex workflows automatically |

## Enabling Sana + Self-Service Agent

1. Sign the Flex Credit Policy
2. Workday sends Security Administrator an in-Workday task
3. Complete the single enablement flow for both Self-Service Agent and Sana
