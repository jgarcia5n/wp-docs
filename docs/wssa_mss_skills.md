# Manager Self Service (MSS) Skills + Tool Groupings

**Status:** Available | **Last Updated:** January 30, 2026

---

## Human Capital Management

**Base**
- View team hire, promotion, birthday, anniversary dates
- View team info and direct reports info
- View worker history
- Get a recommendation for a Change Job template

---

## Workforce Management

**Base**
- View direct reports time off

---

## Talent Management

**Base**
- View team learning status
- View team performance reviews
- View team goals
- View candidate status

**Advanced**
- Start performance review for an employee
- Create and update org goals
- Create goal for multiple employees in bulk

---

## Payroll

**Advanced**
- Request payroll input
