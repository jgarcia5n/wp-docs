# Workday Everywhere – Overview

## What Is Workday Everywhere
Workday Everywhere surfaces Workday functionality inside collaboration tools so employees can complete tasks without switching applications.

**Products:**
- Workday for Microsoft Teams
- Workday for Slack
- Workday Peakon Employee Voice for Microsoft Teams

**Goals:** Meet people in the flow of work · Minimize interruptions · Improve productivity

## Deployment Phases (Workday for Microsoft Teams)
1. Collaborate and Plan
2. Configure, Connect, and Test
3. Communicate, Deploy, and Measure
