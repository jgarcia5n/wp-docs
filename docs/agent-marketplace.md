# Agent Marketplace

## Concept: Agent Marketplace
Part of Workday Marketplace — a catalog of AI agents from Workday and Workday partners designed to address business challenges and extend Workday functionality.

**Access:** https://marketplace.workday.com/en-US/pages/agents

**Who can build/distribute:** Partners in Built on Workday (BoW) or Connect/Platinum partner programs.

**Management:** Register, configure, monitor, and manage agents via ASOR.

### User Types
| User Type | Capabilities |
|-----------|-------------|
| Customers | Discover agents; access features/benefits/responsible AI info; Contact Us for details |
| Workday Partners | Showcase agent solutions; receive customer inquiries via Contact Us (lead generation) |
| Implementers | Search and select agents for business requirements |
| Developers | Upload new agents or skills |

---

## Agent Listing in Workday Marketplace

Changes introduced for agent listings:
- **Agents solution type** – new filter in Workday Marketplace
- **Agent cards and listings** – each agent has a dedicated page with features, benefits, roles, and skill descriptions
- **Categorization** – aligned with Workday's high-level business groupings for consistent search/discovery
- **Agent link** – under AI Solutions section, directs users to agent listings
- **Responsible AI badging** – Workday agents that completed Responsible AI attestation process are badged accordingly
