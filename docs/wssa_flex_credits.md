# Workday Flex Credits – WSSA Consumption

---

## Flex Credits Overview

Flexible, subscription-like credit model for accessing Workday AI features, Illuminate Agents, and platform capabilities.

**Key attributes:**
- Universal flexibility – credits apply to any AI or platform innovation
- Activate/deactivate agents as needs change
- Complimentary base credits included with Workday core product subscription
- Credits only consumed in Production (sandbox = free up to fair use)
- Top up credits anytime
- Rate card provides transparent credit usage rates
- Platform Consumption Console (PCC) provides dashboards for usage/ROI visibility

---

## How WSSA Uses Credits

Credits are consumed **per successful tool call** only.

### Basic Skill Example
- User asks "tell me about my pay information" → agent clarifies → user asks "my next pay date" → **1 credit** (only the successful payslip tool call; clarification question is free)

### Advanced Skill Example
- Actions (e.g., Request Time Off) are only charged **when the action is submitted**, not during the multi-turn clarification flow

### Multiple Tools Example
- A query that calls both the compensation tool and retirement tool = **2 credits** (one per successful tool call)

---

## When Credits Are NOT Used

| Scenario | Reason |
|---|---|
| Guide Skill responses | Not consumption eligible |
| Create Case (ESS – Help Advanced Skill) | Not consumption eligible |
| Conversations with no tool call (e.g., jokes, general chat) | No tool invoked |

## Predicting Credit Need

**Discovery Process**
Workday works with customers to estimate usage based on business scenario. Examples:
- # of candidates screened per year (Recruiting Agent)
- # of contracts ingested per year (Contract Intelligence Agent)
- # of invoices processed per year (Revenue Contracts Agent)
- Platform usage estimate based on previous usage history

**Benchmark Data**
For unknown usage scenarios, credit estimates can be based on Workday benchmark data.
