# Employee Self Service (ESS) Skills + Tool Groupings

**Status:** Available | **Last Updated:** January 30, 2026

---

## Human Capital Management

**Base**
- View total compensation
- View retirement savings
- View bonus history
- View dependents
- View healthcare elections
- View personal information
- View coworker information
- View my org, supervisory org and my team

**Advanced**
- Change personal information (name, marital status, home contact info)
- Request reference letter

---

## Workforce Management

**Base**
- View time off balances
- View time off events
- View time off types
- View holiday calendar
- View time blocks

**Advanced**
- Request time off
- Delete time off
- Edit or correct time off
- Submit my time
- Enter time block
- Check in/out
- Update time block comment

---

## Talent Management

**Base**
- View personal / org goals
- Find jobs / job reqs
- View performance review
- View feedback and view feedback requests
- View talent profile details

**Advanced**
- Create a goal
- Start performance review
- Create a referral

---

## Payroll

**Base**
- View pay information
- View next pay date
- View direct deposit
- View tax forms (W-2)
- View W-4

---

## Help

**Advanced**
- Create a case *(Not consumption eligible)*
